Run project: npm i

Work with git:
 - Clone: git clone https://github.com/tranhaonam291297/hnnews.git
 - Push:
    git add . (add all) or git add <link file>
    git commit -m "type text here"
    git push
 - Pull: git pull
 - Create new branch: git checkout <branch name>
 - Check branch: git status
